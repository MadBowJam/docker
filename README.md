# express-server
Простий сервер, що реалізований за допомогою Node.js та Express.js. Цей сервер надає базовий API для управління користувачами та статтями. Цей проєкт використовує Docker для контейнеризації ізольованих середовищ виконання. Це дозволяє легко встановлювати та запускати проєкт на будь-якому комп'ютері з Docker.

## **Запуск**
### Перед початком роботи з цим проєктом переконайтеся, що у вас встановлено наступне програмне забезпечення:
* Node.js
* Docker
### 1. Встановлення залежностей:
* Відкрийте термінал та перейдіть до кореневої теки проєкт.
* Виконайте команду `npm install` або `yarn install` для встановлення всіх необхідних залежностей.
### 2. Запуск сервера:
* Після встановлення залежностей запустіть сервер командою `docker-compose up`.
* Сервер буде доступний за адресою http://localhost:3000.

## Всі тести проводились через Postman

#### Отримати вітальне повідомлення:
* Метод: GET
* Шлях: /
* Відповідь: Повертає вітальне повідомлення.

#### Отримати всіх користувачів (для перевірки правильності підключення бд):
* Метод: GET
* Шлях: /users
* Відповідь: Повертає список всіх користувачів.
